Lưu Ý


  • Đây là phiên bản thử nghiệm chưa chính thức.

  • Sẽ có lỗi trong khi sử dụng. Vui lòng cân nhắc kĩ

  • Các file patch cũng chưa sắp xếp đúng theo trình tự

  • Nếu bạn muốn cập nhật thêm Patch cho chúng tôi. Vui lòng liên hệ chúng tôi tại đây.

    • Hướng Dẫn Cài Đặt

    - Download phiên bản bất kì của Patch.

    + Với iOS 12 (Hoặc cũ hơn): Sau khi download, chỉ cần nhấn 'Mở trong Filza.'

    + Với iOS 13 (Hoặc cao hơn): Sau khi download, file sẽ lưu trong ứng dụng 'Tệp'.

    - Mở ứng dụng 'Tệp" và tìm file vừa tải, sau đó nhấn giữ và chọn 'Chia sẻ'. Sau đó chọn mở trong Filza.

    - Giải nén file vừa tải ở trong Filza, và vào thư mục vừa giải nén. (Phiên bản 4.2 trở xuống vui lòng chuyển sang bước 5).

    - Đổi tên file lại thành 'patches.plist'.

    - Copy file vừa đổi tên và theo đường dẫn: /var/mobile/Library/Application Support/Flex 3

    - Cuối cùng, dùng iCleaner Pro để dọn rác và tận hưởng!
